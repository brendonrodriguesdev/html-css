Por onde começar?

Existe um trio de tecnologia muito popular para quem tem interesse de aprender programação WEB, a famosa divina trindade da programação WEB como eu gosto de chamar 😅. Ela é composta por 3 tecnologias: HTML, CSS e JS

O que é HTML?

HTML é o acrônimo para: HyperText Markup Language (Linguagem de Marcação de Hipertexto). É uma linguaguem de marcação bastante utilizada nas páginas da internet. HTML não é uma linguagem de programação, mas, sim, uma linguagem de marcação. Para saber mais sobre essa linguagem, clique no link abaxio.👇

o que é CSS?

CSS significa Cascading Style Sheets (Folha de Estilo em Cascata) É uma tecnologia usada para estilizar os sites, colocando cores, imagens, tipos de fontes, fazendo alinhamentos de textos, deixando o site mais bonito e apresentável para o usuário. É importante dizer que, assim como a HTML, as CSS não é uma linguagem de progrmação. É uma linguagem usada para estilização. Para saber mais sobre essa tecnologia, clique no link a baixo. 👇

JS ou JavaScript é a única linguagem de progrmação dessas três tecnologias. É a linguagem de programação mais usada nas páginas de sites da internet e a mais utilizada nos últimos anos.
